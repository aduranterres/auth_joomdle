# Joomdle Auth plugin for Moodle

This plugin is part of Joomdle, a system to integrate Joomla and Moodle.

# Installation

In Moodle, install the auth_joomdle.zip package, included in Joomdle main package, using the web installer.

# Configuration

Follow documentation at https://www.joomdle.com/wiki/Main_Page to get Joomdle working.
